==== 应 DENY(补丁/恢复盲区)====
git apply foo.patch                            -> DENY
      DENY (harness 只读 ADR-0008):git apply / patch 可绕过路径写保护,一律由人工执行。
git apply .claude-runner.patch                 -> DENY
      DENY (harness 只读 ADR-0008):git apply / patch 可绕过路径写保护,一律由人工执行。
patch < x.diff                                 -> DENY
      DENY (harness 只读 ADR-0008):git apply / patch 可绕过路径写保护,一律由人工执行。
git checkout -- .claude/runner.sh              -> DENY
      DENY (harness 只读 ADR-0008):禁止经 git checkout/restore 回滚 harness 路径。
git restore .claude/hooks/stop.sh              -> DENY
      DENY (harness 只读 ADR-0008):禁止经 git checkout/restore 回滚 harness 路径。
git checkout HEAD -- package.json              -> DENY
      DENY (harness 只读 ADR-0008):禁止经 git checkout/restore 回滚 harness 路径。
git checkout -- tsconfig.json                  -> DENY
      DENY (harness 只读 ADR-0008):禁止经 git checkout/restore 回滚 harness 路径。
==== 应 ALLOW(防误杀)====
git checkout main (切分支)                  -> ALLOW
git checkout -- src/App.tsx                    -> ALLOW
git restore src/App.tsx                        -> ALLOW
git commit -m msg                              -> ALLOW
git add -A                                     -> ALLOW
pnpm gate                                      -> ALLOW
rm .claude-runner.patch (清理临时)         -> ALLOW
cat notes.patch | less                         -> ALLOW
